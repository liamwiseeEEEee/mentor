# AI Mentor — build a 3D AI companion from scratch

A free, copy-paste kit for building **Nova** — a 3D animated AI mentor that lives
in your dashboard. One HTML file. No install, no framework, no build tools. Open
it in a browser and it just runs.

Built with AI, no coding background required. The prompts that made it are
included so you can build your own.

---

## 🎮 Try it now

Open **[`ai-avatar.html`](ai-avatar.html)** in your browser (download it, or use
the live link). Three tabs:

- **Avatar + Animations** — a 3D iridescent crystal companion that follows your
  cursor, blinks, and shows emotions. Click any of the 15 animations or 9 face
  expressions to play it on Nova live, and copy the CSS.
- **Chat Box Lab** — a coach-style chat you can type into. Restyle the colors,
  bubbles, thinking animation, and text reveal — then copy it.
- **Cozy Loader** — a playful loading card you can fully customize and copy.

Every piece has a **copy ⧉** button, or use **⬇ Copy everything** to grab the
whole file at once.

## 🚀 Use it in your own project

**Plain HTML site?**
Click **⬇ Copy everything**, save it as `nova.html`, open it. Done.

**React / Next / Vue dashboard?**
Use the small **copy ⧉** buttons inside each lab to grab just the piece you want
(an animation, the chat box, the loader) and paste it into your project. Those
pieces are plain CSS/JS — they drop in anywhere.

> The only part that needs the Three.js line (already baked in) is the 3D crystal
> itself. The animations, chat box, and loader work with no library at all.

## 🛠 Build your own from scratch

1. Open [`all-in-one-prompt.md`](all-in-one-prompt.md) and copy the prompt.
2. Paste it into Claude or ChatGPT.
3. Add features one at a time (see [`HOW-TO-BUILD-NOVA.md`](HOW-TO-BUILD-NOVA.md)
   for the step-by-step order).
4. Save what it gives you as a `.html` file and open it.

The secret: **build in small steps and test each one** before moving on.
